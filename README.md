### **3. –û–ø—Ç–∏–º–∏–∑–∞—Ü–∏—è —Ä–µ–∫—É—Ä—Å–∏–∏**

### **4. –ò—Ç–µ—Ä–∞—Ç–∏–≤–Ω–∞—è –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–∞**
